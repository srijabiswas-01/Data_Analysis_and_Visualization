# 📊 Data Analysis & Visualization: E-Commerce Sales Analytics

An interactive **data analysis and business intelligence dashboard** developed using **Python, Plotly, and Dash** to analyze e-commerce sales performance, customer behavior, and business trends.

## 🚀 Project Overview

This project performs **Exploratory Data Analysis (EDA)** and builds an **interactive visualization dashboard** to extract actionable business insights from e-commerce transactional data.

The dashboard enables users to explore:

* Sales performance analysis
* Profitability trends
* Customer purchasing behavior
* Product category performance
* Geographic sales distribution
* Business KPIs and metrics
